## Dependencies

* [Vagrant](http://vagrantup.com)
* [VirtualBox](http://virtualbox.org)

## Instalation & usage

```shell
git clone https://github.com/suissa/workshop-be-mean-beer-crud
cd workshop-be-mean-beer-crud
vagrant up
vagrant ssh
npm start
```

Now your app should be up and running at http://192.168.33.10:3000
